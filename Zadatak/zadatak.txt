Kreirati novi Java projekt koji će koristiti radne okvire Hibernate i JPA
i implementirati primjer veze „M:N” s
entitetima „Tvrtka” (engl. Company) ,
"Osoba” (engl. Person)
i „Ugovor” (engl. Contract).

Klasa „Person” mora sadržavati
identifikator „Long id” i naziv
„String name”
Set objekata klase „Contract”
anotacije „@ManyToMany” i „@JoinTable”.

Klasa „Company” mora sadržavati
identifikator „Long id”
naziv „String name”
Set objekata klase „Contract” kojem pripada taj sastojak označen s anotacijama „@ManyToMany” i „@JoinTable”.

Klasa „Contract” mora sadržavati
 identifikator „Long id”,
 varijablu „LocalDate” koji označava datum početka ugovora,
 cjelobrojnu vrijednost koja označava trajanje ugovora,
 „BigDecimal” vrijednost koja predstavlja iznos plaće
 Set osoba i Set kompanija označenih s „@ManyToMany” anotacijama.
